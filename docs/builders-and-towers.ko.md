# 빌더와 타워

이 문서는 현재 코드에 등록된 빌더와 타워 계열을 정리합니다. 기준 코드는 `JobRegistry`와 `ProductionTowerCatalogs.reloadBuiltIns(...)`입니다. 타워 수치, 특수 효과, 등록 API는 [프로덕션 타워 카탈로그](production-tower-catalog.ko.md)를 봅니다.

## 빌더 목록

| 직업 ID | 표시명 | 컨셉 | 타워 계열 |
|---|---|---|---|
| `semion-td:recruit` | 무직 | 기본 직업입니다. 실전용 빌더로 보지 않습니다. | 없음 |
| `semion-td:villager_towers` | 주민 빌더 | 타워를 오래 살리고 업그레이드해 누적 보너스를 받습니다. | 주민 |
| `semion-td:villager_adv_towers` | 주민 ADV 빌더 | 경험치와 평판을 쓰는 주민 리워크 계열입니다. | 주민 ADV |
| `semion-td:undead_towers` | 언데드 빌더 | 흡혈, 탱킹, 성장형 원거리/근거리 분기를 씁니다. | 언데드 |
| `semion-td:animal_towers` | 동물 빌더 | 같은 계열 타워를 모아 스택 보너스를 받습니다. | 동물 |
| `semion-td:warlock_towers` | 흑마법사 | 아군 타워 희생과 전용 코어 타워를 씁니다. | 흑마법사 |
| `semion-td:legion_towers` | 무리 빌더 | 여러 저가 계열과 환영 타워를 조합합니다. | 무리 |
| `semion-td:resonance_towers` | 무블룸 빌더 | 서로 다른 무블룸을 붙여 공명 단계를 올립니다. | 무블룸 |
| `semion-td:illager_towers` | 우민 빌더 | 습격 게이지를 모아 우민 타워 전체를 강화합니다. | 우민 |

## 타워 계열

### 주민

- 직업 ID: `semion-td:villager_towers`
- 표시명: 주민 빌더
- 흐름:
  - 주민 원거리 기본 타워 -> 사서 타워 -> 성직자 타워
  - 눈 골렘 타워 -> 라마 타워 -> 철 골렘 타워
  - 알레이 타워 -> 알레이 타워(강함) -> 갑옷 제조인 타워
  - 알레이 타워 -> 대장장이 타워 -> 강화 대장장이 타워
  - 오셸롯 타워 -> 저격 캣 타워 -> 강화 저격 캣 타워
  - 오셸롯 타워 -> 라클 캣 타워 -> 강화 라클 캣 타워

### 주민 ADV

- 직업 ID: `semion-td:villager_adv_towers`
- 표시명: 주민 ADV 빌더
- 흐름: 주민 빌더와 비슷한 분기 구조를 쓰지만 별도 타워 ID를 가진 별개 빌더입니다. 타워 ID에는 `villager_adv_` prefix가 붙습니다.
- 자원: 각 타워는 경험치를 사용하고, 빌더 전체는 평판 보너스를 받습니다.

### 언데드

- 직업 ID: `semion-td:undead_towers`
- 표시명: 언데드 빌더
- 흐름:
  - 좀비 타워 -> 허스크 타워 -> 드라운드 타워
  - 뼈 타워 -> 보그드 타워 -> 스트레이 타워
  - 뼈 타워 -> 위더 스켈레톤 타워 -> 강화 위더 스켈레톤 타워
  - 좀비 말 타워 -> 스켈 말 타워

### 동물

- 직업 ID: `semion-td:animal_towers`
- 표시명: 동물 빌더
- 흐름:
  - 돼지 타워 -> 돼지 타워 -> 돼지 타워
  - 늑구 타워 -> 재빠른 늑구 타워 -> 개빠른 늑구 타워
  - 토끼 타워 -> 토끼 타워 -> 토끼 타워
  - 사냥 여우 타워 -> 붉은 여우 타워 -> 설원 여우 타워

### 흑마법사

- 직업 ID: `semion-td:warlock_towers`
- 표시명: 흑마법사
- 흐름:
  - 흑마법사 기본 타워 -> 원거리 흑마법사 타워
  - 흑마법사 기본 타워 -> 근접 흑마법사 타워
  - 희생 타워 -> 희생"양" -> 희생"양"
  - 애완 개구리 -> 애완 개구리 -> 애완 개구리
- 제한: 흑마법사 코어는 한 라인에 하나만 둘 수 있습니다.

### 무리

- 직업 ID: `semion-td:legion_towers`
- 표시명: 무리 빌더
- 흐름:
  - 닭 계열 -> 인싸 닭 타워 또는 아찐 닭 타워
  - 슬라임 계열 -> 슬라임 타워
  - 펭귄 계열 -> 강화 땡컨타워
  - 앵무 계열 -> 앵무 타워
  - 염소 계열 -> 짱쌘 염소 타워 -> 개쌘 염소 타워
  - 환영 타워는 별도 starter로 등록됩니다.
- 참고: 벌 계열은 `TowerBalanceConfig` 기본값에 수치가 있지만 현재 카탈로그 등록 코드는 비활성 주석 상태입니다.

### 무블룸

- 직업 ID: `semion-td:resonance_towers`
- 표시명: 무블룸 빌더
- 흐름:
  - 집속: Focus Crystal -> 해바라기 집속 무블룸 -> 주황 튤립 초점 무블룸
  - 파동: Wave Crystal -> 파란 난초 진동 무블룸 -> 푸른 들꽃 파동 무블룸
  - 서리: Frost Crystal -> 하얀 튤립 서리 무블룸 -> 데이지 빙결 무블룸
  - 증폭: Amplify Crystal -> 라일락 정원 무블룸 -> 작약 공명 무블룸
- 조건: 다른 종의 무블룸을 1칸 안에 모으면 공명 단계가 올라갑니다.

### 우민

- 직업 ID: `semion-td:illager_towers`
- 표시명: 우민 빌더
- 흐름:
  - 변명자 타워 -> 변명자 대장타워 -> 파괴수타워
  - 약탈자 타워 -> 약탈자 대장타워(단일) -> 소환사타워(단일)
  - 약탈자 타워 -> 약탈자 대장타워(광역) -> 소환사타워(광역)
  - 벡스 타워 -> 마녀타워(약자 표식) -> 환술사타워(약자 표식)
  - 벡스 타워 -> 마녀타워(강자 표식) -> 환술사타워(강자 표식)
- 자원: 웨이브 처치, 인컴 처치, 표식 처치, 우민 타워 사망으로 습격 게이지를 얻습니다.

## 실제 노출 기준

카탈로그에 등록된 타워라도 선택한 빌더의 `canUseTower(...)`가 허용해야 플레이어가 설치할 수 있습니다. `/semiontd tower list`와 타워 UI는 현재 빌더가 사용할 수 있는 starter 타워만 보여줍니다.
